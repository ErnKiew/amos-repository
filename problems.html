The AMOS Repository - Problems

All problems will be formatted as such: "Name (Source) - Difficulty"

CB --> Codebreaker, CF --> Codeforces

Some problems will have spoiler tags that may accidentally serve as hints to the question. We recommend at least thinking through the problem before pressing it.

  • Eeducation SUBTASK 2 only (CB) - 3/10 Sidenote: If you've solved before with STL Data structures, try solving subtask 2 without additional data structures.
  • Smurf (CB) - 3.5/10 Sidenote: Think hard! Search for patterns in the testcases. Generate your own if it helps.
  • Explosion (CB) - 3.5/10 Sidenote: You must use pair/custom sorting. Go learn that beforehand. (ask the programming heads!)
  • Illuminate (CB) - 5/10 Sidenote: Solve subtasks first. Likely you will not be able to find the optimal answer first try!
